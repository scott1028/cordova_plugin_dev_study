# How to Install and test your own plugin, in your cordova project type this command:
    cordova plugin add ..\cordova_plugin_dev

# Cordova Plugin Project Structure
    src/{{Native Code Implement}}
    www/{JavaScript Interface{}}
    plugin.xml
